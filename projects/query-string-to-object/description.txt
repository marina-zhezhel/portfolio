Напиcать функцию parse(string), которая принимает аргументом query string, и возвращает объект. Типы данных, которые должны распознаваться: строки, числа (только целые), булевы.

parse(''); // -> {}
parse('test=10'); // -> {"test": 10}
parse('test=bar'); // -> {"test": "bar"}
parse('test=10&foo=bar'); // -> {"test": 10, foo: "bar"}
